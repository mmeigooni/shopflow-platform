# ShopFlow Platform

ShopFlow Platform is a mock eCommerce backend mono-repo used to generate realistic pull-request history for ingestion and memory encoding demos.

## Services
- `services/catalog`
- `services/checkout`
- `services/orders`
- `services/payments`

## Shared libraries
- `shared/auth`
- `shared/db`
- `shared/logging`

This repository is intentionally lightweight and focuses on PR/review realism rather than production runtime behavior.
